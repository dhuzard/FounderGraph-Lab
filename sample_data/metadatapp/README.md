# Metadatapp Sample Dataset

Metadatapp is a fictional B2B SaaS startup for testing FounderGraph Lab. It helps small data teams inventory spreadsheets, BI dashboards, and warehouse tables, then flags stale or risky metadata before audits.

The dataset is intentionally compact and synthetic. It includes a venture graph, assumptions, evidence, risks, and source notes suitable for Qdrant indexing and export smoke tests.
