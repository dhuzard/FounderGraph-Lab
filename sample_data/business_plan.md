# Metadatapp Business Plan

Metadatapp will start with paid pilots for preclinical CROs that run repeated study workflows and need cleaner metadata capture.

The first wedge is a lightweight metadata layer that maps experimental context across study planning, execution, and reporting tools.

Assumptions:

- CROs will pay for integration if it reduces duplicated data entry.
- Pharma translational teams will sponsor pilots when metadata quality affects AI readiness.
- Compliance reporting pressure creates urgency for academic animal facilities.

Milestones:

- Complete three CRO discovery interviews.
- Launch one paid metadata mapping pilot.
- Demonstrate a reusable evidence matrix for virtual control group readiness.
